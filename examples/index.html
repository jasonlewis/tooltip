Tooltip is a very simple yet elegant tooltip plugin for jQuery. You can attach tooltips to anything using the power of jQuery. Tooltips can also be customized to show at different positions of the element.
  1. Top (default behaviour)
  2. Top Left
  3. Top Right
  4. Right
  5. Left
  6. Bottom Left
  7. Bottom Right
  8. Bottom

As well as positioning you can set it to attach to elements added to the DOM by jQuery. Check it out...
